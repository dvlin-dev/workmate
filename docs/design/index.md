## 关键不变量（实现时反复回看）

- 进度流 `ProgressEvent` 是周报唯一原料，每次录入/归因都落事件。
- `weekOf` 以周一为锚，跨周自动新建 `WeeklyPlan`。
- 看板是 agent 替用户维护的**产物**，不是手动拖拽对象——这是与普通效率软件的根本区别。
- 唯一输入方式是"说话"；便利贴输入框与对话是同一进度流的两种皮肤。
